IPSherlock - "Check My IP" Design Options

Option 1: Toggle Switch

Search IP/Domain Investigate My IP

Pros

  • Clear separation of functionality
  • Intuitive mode switching
  • Modern UI element
  • Prominent but not distracting

Cons

  • Takes up vertical space
  • Might be unfamiliar to some users
  • More complex to implement

Option 2: Subtle Link

Investigate my own digital footprint

Pros

  • Doesn't compete with main search
  • Elegant and subtle design
  • Detective-themed language
  • Positioned for easy discovery

Cons

  • Could be overlooked by some users
  • Text might be too long
  • Right-alignment might feel disconnected

Option 3: Example Link

Examples: 8.8.8.8, google.com, or trace my own IP

Pros

  • Contextually relevant placement
  • Feels like a natural suggestion
  • Minimal design impact
  • Concise wording

Cons

  • Could be missed among examples
  • Less prominent than other options
  • Might be confused with example inputs
