Understanding IP Addresses & Domains

What is an IP Address?

An Internet Protocol (IP) address is a unique numerical label assigned to each device connected to a computer network. Think of it as a home address for your device on the internet. IP addresses serve two main purposes: identifying the host or network interface, and providing the location of the device in the network.

There are two versions of IP addresses in use today:

  • IPv4: The original system using 32-bit addresses (e.g., 192.168.1.1)
  • IPv6: The newer system using 128-bit addresses (e.g., 2001:0db8:85a3:0000:0000:8a2e:0370:7334)

What is a Domain Name?

Domain names are the human-readable addresses of websites. They translate the numerical IP addresses into memorable names that are easier for people to use. For example, "google.com" is much easier to remember than "142.250.190.78".

Domain names are organized in a hierarchical structure:

  • Top-Level Domain (TLD): The rightmost part (.com, .org, .net)
  • Second-Level Domain: The name chosen by the website owner (google in google.com)
  • Subdomains: Additional prefixes (mail in mail.google.com)

Why Investigate IP Addresses and Domains?

Understanding the details behind IP addresses and domain names is valuable for many reasons:

  • Security Analysis: Identifying potentially malicious sources of traffic
  • Network Troubleshooting: Diagnosing connection issues between systems
  • Digital Forensics: Tracing online activities for investigations
  • Website Ownership Research: Discovering who operates a particular website
  • Geolocation Verification: Confirming the physical location of servers or users
